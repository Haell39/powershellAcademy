Introdução ao PowerShell

PowerShell é uma shell de linha de comando e linguagem de script... e a várias formas de se chegar ao mesmo resultado:

> Write-Host "Hello, World!"
# OU
> Write-Output "Hello, World!"

Comandos Básicos

Outros exemplos de aliases e funções semelhantes é:

# Voce tambem pode usar Echo e Write para imprimir na tela
> Echo "Hello, PowerShell!"
> Write "Hello, Testing!"
# Outro exemplo disso são os seguintes comando:
> dir
> ls
> Get-ChildItem
# Ambos servem para listar os arquivos e diretórios do diretório atual

Criação do seu proprio comando (Alias)

# Por exemplo voce pode criar um alias para o comando "ping" do cmd usado para testar a conexão com um servidor
> Set-Alias -Name ping -Value Test-NetConnection
# Agora é possível usar o comando "ping" para testar a conexão ao invés de "Test-NetConnection"

OBS:
# Se o alias já estiver em uso, fique atento pois será sobrescrito !!

# O alias funciona até o termino da seção, para evitar ficar recriando é possível importal todos seus aliases doe um Excel antes de comçar a trabalhar
